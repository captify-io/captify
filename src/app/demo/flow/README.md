# Flow Package Demo

This demo showcases all capabilities of the `@captify/flow` package.

## Demo Sections

### 1. Node Types
Demonstrates all available node shapes:
- **Default Node**: Standard rectangular node
- **Circle Node**: Circular node with icon support
- **Rounded Rectangle**: Rounded corners rectangle
- **Diamond**: Decision/conditional node shape
- **Parallelogram**: Input/output node shape
- **Arrow Rectangle**: Directional arrow node

### 2. Edge Types & Styles
Shows different edge configurations:
- Animated edges
- Static edges
- Labeled edges
- Workflow edges
- Custom styling

### 3. Toolbar & Controls
Interactive toolbar features:
- Save functionality
- Undo/Redo (when implemented)
- Auto-layout
- Clear canvas
- Zoom controls

### 4. Node Palette
Drag-and-drop node creation:
- Categorized nodes (action, decision, process)
- Searchable palette
- Icon preview
- Quick add functionality

### 5. Sidebar Configuration
Node and edge property editing:
- Node properties panel
- Edge configuration
- Custom property fields
- Real-time updates

### 6. Workflow Mode
Pre-configured workflow canvas:
- Workflow-specific nodes
- Status indicators
- Execution flow visualization
- Step-by-step process

### 7. Custom Configuration
Fully featured example:
- All features enabled
- Auto-save functionality
- Complex node graph
- Real-world use case

## Features Demonstrated

✅ All 6 node types (default, circle, roundedRect, diamond, parallelogram, arrowRect)
✅ Multiple edge types and styles
✅ Toolbar with save and controls
✅ Node palette with drag-and-drop
✅ Sidebar for node/edge configuration
✅ Workflow mode
✅ Custom configurations
✅ Auto-save functionality
✅ Interactive examples
✅ Real-time updates
